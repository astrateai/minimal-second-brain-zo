# Pilot Success Scorecard (Week 1)

Use this for wife/brother pilot validation.

## Minimum pass criteria

- [ ] Client added at least 3 files to `raw/`
- [ ] `./.ops/bin/compile` ran successfully at least once
- [ ] `wiki/index.md` exists and includes topic links
- [ ] At least 1 question was answered and saved to `outputs/`
- [ ] `./.ops/bin/weekly-digest` generated weekly digest file

## Quality checks

- [ ] `./.ops/bin/health-check` reports no missing source references
- [ ] No obvious duplicate topic pages
- [ ] Client understood the single rule: "I only add files to raw/"

## Engagement checks

- [ ] Client responded to at least one weekly nudge
- [ ] Client added at least one new source after first digest

## Decision

- Pass: all minimum pass criteria complete.
- Needs revision: any minimum pass criterion failed.
